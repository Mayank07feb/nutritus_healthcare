# Nutritus Healthcare

A modern web application built using **HTML, CSS (Tailwind), EJS, and JavaScript** for showcasing products, blogs, lab reports, and contact information for **Nutritus Healthcare Pvt. Ltd.**

---

## Features

- Responsive **header and navigation** with active link highlighting.
- **Hero section** with background image and product carousel.
- Dynamic **Products, Blogs, Lab Reports, and Contact** pages.
- Mobile-friendly design with Tailwind CSS.
- Image carousel using **Swiper.js** with autoplay and responsive slides.
- Clean and modern UI/UX.
